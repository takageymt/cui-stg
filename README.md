CUI-Shooter
===

Shooting game running on a terminal.

![screenshot1]()
![screenshot2]()

Requirements
===

- gcc
- curses.h or ncurses.h

How to Play
===

Copy `main.cpp` to your directory and then execute:

     $ g++ -lcurses main.cpp -o cui-stg
     $ ./cui-stg

And you follow instructions on the window.
